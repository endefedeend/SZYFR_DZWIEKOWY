Program do szyfrowania tekstu w pliku audio. 
Zasada działania jest prosta, nadawca (szyfrowanie.py ) i odbiorca
(odszyfrowanie.py) posiadają tajny klucz(jest to przypisanie odpowiedniej 
częstotliwości do liter).Plik audio jest zaś kluczem nietajnym. 
Dzięki wiedzy o czestotliwości dla danej litery, odbiorca jest w stanie 
zrekonstruować oryginalną treść wiadomości tekstowej.
